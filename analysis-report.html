Evaluation

技术评价

项目亮点与可改进之处的客观分析。

✦ 设计亮点

  • C/Python 混合架构 — C 引擎负责图形渲染和数学计算，Python 负责灵活的数据采集
  • 双数据源互备 — Gate.io + OKX 双重保障，Gate.io 国内可直连
  • 全链路增量更新 — 所有脚本均按日期检测，只拉取/计算新数据
  • 通达信兼容 — 线性回归公式与通达信软件一致，X 从 1 开始编号
  • 极致性能 — 毫秒级启动，~5-10MB 内存占用，~200KB 二进制体积
  • 多周期切换 — 运行时点击切换日线/月线/季线，自动重新计算通道

⚡ 可改进处

  • 无单元测试 — 整个项目没有任何测试文件
  • system() 调用脆弱 — Python 进程通过 system() 启动，无错误传递和状态反馈
  • 硬编码路径 — 多处硬编码 Windows 路径 (C:\raylib, D:\openskills)
  • 无配置系统 — 所有参数 (窗口大小、颜色、数据路径) 编译时硬编码
  • 单线程渲染 — raylib 渲染循环运行在主线程，大数据量时可能掉帧
  • Python 依赖 — 数据更新依赖 Python 环境 + Playwright + requests

Project Relations

项目血缘关系

BTC/ (Python 原型) 是 BTC-C 的前身，使用 Matplotlib + Pandas 实现基础 K 线和回归通道分析。 BTC-C 是完整的 C 语言重写+增强版：将可视化从 matplotlib 迁移到 raylib 硬件加速渲染， 增加链上指标 (MVRV Z-Score, CVDD) 系统，支持多周期切换 (日/月/季线)， 引入 Python 数据管道实现自动更新。 性能提升：启动时间从 ~2-3s 降至毫秒级，内存占用从 100MB+ 降至 5-10MB。
